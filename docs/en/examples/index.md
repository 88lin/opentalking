# Examples

Scenario recipes answer how OpenTalking can be used in real business flows.

## Huangshan Scenic Area Knowledge Digital Human

Use the deployed OpenTalking image to build a Huangshan Scenic Area knowledge digital human for realtime knowledge-base Q&A and offline explainer video generation.

- Guide: [OpenTalking Huangshan Scenic Area Knowledge Digital Human Operation Guide](huangshan-digital-human-guide.md)

## WeChat Chat History Import

Use WeFlow to export WeChat chat history, import it into OpenTalking, and generate a digital-human persona and memory library for realtime conversation testing.

- Guide: [OpenTalking WeChat Chat History Import Case Study](Wechat_import.md)
